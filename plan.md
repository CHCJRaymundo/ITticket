# IT Ticketing System — Vibe Coding Plan

## Overview
Build a complete internal IT ticketing system with Flask + SQLite + HTML/CSS/JS.

## Key Features
1. **Ticket Submission & Viewing** — Submit tickets with title, description, priority, category; view own ticket statuses
2. **Team Status Dashboard** — Public page showing who's available/free and what each member is actively working on
3. **File Upload/Download** — Attach images/documents to tickets; IT staff can view/download them
4. **Admin Panel** — IT staff can view all tickets, update status, assign to team members
